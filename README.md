# SWF2023 블록체인 스터디
SWF2023 해커톤 대비를 위한 web3.0, 블록체인 관련 스터디용 저장소입니다.
스터디는 7월간 매주 최소 1회씩 직전주차 스터디에서 합의된 주제, 일정, 장소에 따라 진행합니다.

내주 합의된 사항에 따라 추가 스터디 일정을 기획할 수 있으며 비대면 회의 수단을 유연하게 사용할 수 있습니다.
